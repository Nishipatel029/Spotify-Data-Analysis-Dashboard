# 🎧 Spotify Data Analysis Dashboard

An interactive Power BI dashboard to analyze Spotify streaming data and uncover insights on track performance and user engagement.

##  Dashboard Preview
<img width="1286" height="716" alt="image" src="https://github.com/user-attachments/assets/c597e940-becf-45a4-ab83-8656d4725ecd" />


## Key Features

* Interactive filters (Artist, Track, Release Year)
* Stream trend analysis and KPI metrics
* Track popularity and usage patterns
* Heatmap visualization for listening behavior
* Integration with Spotify API for enriched data

---

## Tech Stack

* Power BI
* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Spotify API
